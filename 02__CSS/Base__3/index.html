Блок навигации.

  • ширина 15 ем
  • должен присутствовать цвет фона
  • располагается слева
  • блок навигации наполните контентом достаточным чтобы по высоте он начинал захватывать вторую статью, как показано в данном примере

Статья 1

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех статьях сделайте отступ снизу от соседей равным 2ем, при помощи свойства
margin-bottom

Статья 2

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех статьях сделайте отступ снизу от соседей равным 2ем, при помощи свойства
margin-bottom Картинка вставлена при помощи тега, плавание организовано по правому краю
данной статьи для этой картинки
Нижний блок подвал. Занимает всю доступную ширину. Высота определяется помешенным внутрь контентом.
