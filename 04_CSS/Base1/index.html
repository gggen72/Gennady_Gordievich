Форматы файлов.

.DOC или .doc (от англ. document) - расширене имени файла, используемого для файлов,
представляющих текст, с разметкой или без. Расширение .DOC часто использовалось для
обозначения простых текстовых файлов без формирования, однако позже стало
использоваться для двоичных форматов с разметкой.
Документ MS Word

Portabl document format (PDF) - межплатформенный формат электронных документов,
разработанный фирмой Adobe Systems с тспользованием ряда возможностей языка
Postscript.
Документ PDF

ZIP - популярный формат сжатия данных и архивации файлов. Файл в этом формате
обычно имеет расширение .zip и хранит в сжатом или несжатом виде один или несколько
файлов, которые можно из него извлечь путем распаковки с помощью спецальной
программы.
